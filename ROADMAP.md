# 🚀 Roadmap de Futuras Implementações - PratesPaiva Core

Este documento registra as ideias e melhorias sugeridas para evoluções futuras do sistema, conforme discutido com o usuário.

## 🛠️ Módulo de Ferramentas (Inventário Avançado)
- [ ] **Alocação Inteligente**: Vincular ferramentas específicas a uma Obra/OS ativa para saber exatamente onde cada item está em tempo real.
- [ ] **Controle de Manutenção**: Alertas baseados em tempo de uso ou data para revisão técnica de máquinas (ex: troca de carvão, lubrificação).
- [ ] **Histórico de Uso**: Log de quem utilizou a ferramenta por último e em qual obra.

## 💰 Inteligência Financeira & Custos
- [ ] **Lógica de "Dinheiro Líquido"**:
  - [ ] Registro de despesas de materiais por obra.
  - [ ] Abatimento automático de custos de quebra de ferramentas no lucro do serviço.
  - [ ] Relatório de margem de contribuição real por tipo de serviço.
- [ ] **Fluxo de Caixa Projetado**: Visualização de recebíveis futuros baseados em datas de entrega de obras.

## 📄 Gestão de Documentos & NFs
- [ ] **Leitura de NF-e**: Upload de XML/PDF de Notas Fiscais para preenchimento automático de valores no financeiro.
- [ ] **Repositório de Contratos**: Area para armazenamento de contratos assinados por cliente/condomínio.
